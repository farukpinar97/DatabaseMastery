TransportMongoDb, modern web mimarisi ile geliştirilen, kargo ve lojistik firmaları için kapsamlı bir yönetim ve takip platformudur.

---

## 📋 Proje Hakkında

TransportMongoDb, kullanıcıların kargo takip numarası ile gönderilerini anlık olarak takip edebildiği; yöneticilerin ise kargo oluşturma, hareket kaydı ekleme ve tüm süreci admin panel üzerinden yönetebildiği full-stack bir web uygulamasıdır.

Frontend'de kargo takip sayfası ve lojistik firmaya ait tanıtım içerikleri yer alırken; admin panelde kargo yönetimi, hareket geçmişi yönetimi ve site içerik yönetimi gibi kapsamlı araçlar sunulmaktadır.

---

## 🎯 Öne Çıkan Özellikler

### 🌐 Frontend (Kullanıcı Tarafı)
- Takip numarası ile anlık kargo sorgulama
- Kargo hareket geçmişi timeline görünümü
- Slider, marka, hizmet ve referans tanıtım sayfaları
- Nasıl Çalışır ve SSS sayfaları

### 🔧 Admin Panel
- **Kargo Yönetimi** — Kargo oluşturma, düzenleme, silme; gönderici/alıcı bilgileri, rota yönetimi
- **Hareket Yönetimi** — Kargo hareketi ekleme, düzenleme, silme; konum, tarih ve durum takibi (`Gönderi Alındı / Transfer Merkezinde / Yolda / Dağıtımda / Teslim Edildi`)
- **İçerik Yönetimi** — Slider, marka, hizmet, hakkımızda, referans, proje portföyü, SSS yönetimi
- **ViewComponent Mimarisi** — Header, footer ve tekrar eden alanlar component bazlı yapı

---

## 🛠️ Kullanılan Teknolojiler

| Teknoloji | Açıklama |
|---|---|
| ASP.NET Core 9 | Web framework, MVC mimarisi |
| MongoDB | NoSQL veritabanı, tüm veri depolama |
| AutoMapper | Entity ↔ DTO dönüşümleri |
| Bootstrap | Responsive tasarım |
